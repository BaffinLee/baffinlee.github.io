# Baffin Lee's blog

> check it out [https://baffinlee.com](https://baffinlee.com)

## setup
```bash
# install
npm run init
```

## write blog
```bash
# generate a new post template
npm run new:post
# or generate a new page template
npm run new:page
# publish it
npm run publish
# preview
npm run preview
```
